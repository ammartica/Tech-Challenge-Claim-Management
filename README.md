# Tech-Challenge-Claim-Management

Description:

Haipriori Technical Challenge
This repository exists as a technical challenge for an interview with HAIPRIORI. The goal is to design, implement, and deliver a small Medical Center Claim Management feature.

It is a very simple, three page app that has authentication, CSV file import/export features, and visualization of said data.

The stack is as follows:

Backend: Docker + Ruby on Rails
Frontend: ReactJS + Vite
Database: PostgreSQL
UI: Ant Design

Installation:

download seed (staff and admin user) using:
      docker compose exec backend bundle exec rails db:seed
